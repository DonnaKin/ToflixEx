### 구조 설명
```
Atoms
    - 이 이상 나눌 수 없는 단위
    - ex 버튼, 텍스트 입력란, 라벨

Molecules
    - Atoms를 여러개 합친 것
    - Organisms 내 들어가는 항목
    - ex 입력폼(라벨 + 입력란), Item vue, header, footer

Organisms
    - 1개의 명확한 기능을 가진 것
    - ex list, Container

Templates
    - 페이지의 틀을 구성하는 것, Organisms 집합체
    - 와이어 프레임

Pages (사용안함 / Views로 대체하거나 필요할 때 생성)
    - 페이지에 컨텐츠가 있는 것, Templates 집합체
    - 디자인 완성 견본
```
